# Исследование средней сложности алгоритма Евклида и гипотезы Портера

Данный проект посвящен изучению «типичной» (средней) сложности алгоритма Евклида. В то время как наихудший случай алгоритма описывается теоремой Ламе (числа Фибоначчи), среднее количество шагов связано с более глубокими разделами теории чисел и анализа.

## Теоретическая основа

### Теорема Хейльбронна-Диксона
Известно, что среднее число шагов $T(n)$ для алгоритма Евклида, примененного к числам, меньшим $n$, асимптотически определяется формулой:
$$T(n) \approx \frac{12 \ln 2}{\pi^2} \ln n \approx 0.843 \ln n + C$$
Этот результат был установлен Г. Хайльбронном.

### Связь с цепными дробями
Алгоритм Евклида эквивалентен разложению рационального числа в правильную цепную дробь. Каждое преобразование в алгоритме соответствует нахождению частичного отношения (неполного частного) $A_{n+1} = \lfloor 1/X_n \rfloor$. Дональд Кнут в своей работе «Искусство программирования» детально рассматривает эту взаимосвязь.

### Гипотеза Портера (1975 г.)
Гипотеза Портера уточняет асимптотическую формулу, утверждая, что остаточный член стремится к нулю:
$$T(n) = \frac{12 \ln 2}{\pi^2} \ln n + C + o(1)$$
Константа Портера $C$ имеет точное выражение через производную дзета-функции Римана в точке 2.

## Реализация

Проект включает в себя программную реализацию алгоритма на языке Python с использованием метода Монте-Карло для оценки среднего количества шагов.

### Основные функции:
*   `AE_steps_counter(a, b)`: Подсчитывает количество делений с остатком. Важной особенностью является то, что при вводе пары, где первое число меньше второго, алгоритм совершает на один шаг больше для их перестановки.
*   `avg_AE_steps_counter(n, samples)`: Вычисляет среднее количество шагов для случайных пар чисел $(a, b) \le n$.

## Численный эксперимент

В рамках работы были проведены два типа экспериментов для значений $n$ от $10^3$ до $10^6$:

1.  **Тест с малой выборкой (50,000 сэмплов):** Результаты усреднены по 100 итерациям. Полученный угловой коэффициент (slope) составил в среднем **~0.8422**.
2.  **Тест с большой выборкой (5,000,000 сэмплов):** Высокая точность подтвердила теоретический коэффициент. Полученные значения варьировались в диапазоне **0.8420 – 0.8423**.

График зависимости $T(n)$ от $\ln(n)$ подтверждает близость экспериментальных данных к теоретическому значению коэффициента $0.843$, установленному Хейльбронном-Диксоном.
