# Исследование средней сложности алгоритма Евклида и гипотезы Портера

Данный проект посвящен программному исследованию «типичной» (средней) сложности алгоритма Евклида и верификации гипотезы Портера. 

## Теоретическая база

### 1. Теорема Хейльбронна-Диксона
В отличие от наихудшего случая (числа Фибоначчи, теорема Ламе), среднее количество шагов $T(n)$ для случайных пар чисел $(a, b) \le n$ асимптотически определяется формулой:
$$T(n) \approx \frac{12 \ln 2}{\pi^2} \ln n \approx 0.843 \ln n + C$$

### 2. Связь с цепными дробями
Алгоритм Евклида эквивалентен разложению рационального числа $v/u$ в правильную цепную дробь. Количество шагов алгоритма в точности совпадает с количеством **частичных отношений** (неполных частных) в разложении.

### 3. Гипотеза Портера (1975 г.)
Гипотеза уточняет асимптотическое поведение, утверждая, что остаточный член $o(1)$ стремится к нулю:
$$T(n) = \frac{12 \ln 2}{\pi^2} \ln n + C + o(1)$$
Константа Портера $C$ выражается через производную дзета-функции Римана в точке 2.

## Особенности реализации

Проект оптимизирован для проведения массивных статистических вычислений:
*   **Метод Монте-Карло:** Позволяет избежать полного перебора пар $n^2$ и проводить оценку на выборках большого объема.
*   **Ускорение Numba:** Применение декораторов `@njit` и `parallel=True` позволило сократить время обработки **5 000 000 образцов** для каждого $n$ до нескольких секунд.
*   **Систематическая погрешность:** Программа учитывает, что при вводе пары $a < b$ алгоритм совершает на один шаг больше для перестановки аргументов.

## Функционал
Программа имеет интерактивное меню:
1. **Расчет для конкретного $n$:** вывод среднего числа шагов, теории и погрешностей.
2. **Визуализация:** построение графика $T(n)$ от $\ln n$ с аппроксимацией методом наименьших квадратов.
3. **Проверка гипотезы Портера:** вычисление экспериментальной константы и анализ сходимости.
